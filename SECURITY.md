**Project:** DGB Sentinel AI  
**Author:** DarekDGB  
**License:** MIT

## Reporting a Vulnerability

If you believe you found a security issue:

1. **Do not publish exploit details in a public issue.**
2. Prefer GitHub Security Advisories (private reporting), if available.
3. Otherwise, open an issue marked **SECURITY** with minimal details and wait for next steps.

## What to include

- Impact: what an attacker could achieve
- Steps to reproduce (safe / non-destructive if possible)
- Logs or screenshots that support the claim

## Disclosure

This repo aims to fix validated issues quickly without publishing weaponized details before a patch exists.
